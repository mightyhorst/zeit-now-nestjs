## Installation

```bash
$ npm install
```

## Running the NestJS Application

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Deploying to Zeit Now

First we need to ensure we build our NestJS application into the `/dist/` folder, and then we can deploy it with `now`! Simple :)

```bash
$ npm run build && now
```
